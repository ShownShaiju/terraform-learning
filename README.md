# Terraform Learning

A hands-on learning journey documenting Terraform concepts from first principles.

## Contents

- **Day-01-Basics.md** - Core Terraform concepts including Infrastructure as Code, workflow, providers, resources, state management, and remote state configuration.

## Objective

This repository captures learning progress with a focus on understanding Terraform by asking questions and building mental models rather than just memorizing syntax.

## Key Topics Covered

- What problem Terraform solves
- Terraform workflow (init → plan → apply → destroy)
- Providers and cloud platform integration
- Resources and infrastructure definition
- Terraform state and state management
- Remote state configuration with S3
- File structure and configuration formats

## Getting Started

1. Read through the markdown files to understand Terraform concepts
2. Refer to the [Terraform Documentation](https://www.terraform.io/docs) for detailed reference
3. Practice by creating your own configurations based on the concepts learned

## Notes

These notes emphasize the thinking process and reasoning behind Terraform design patterns rather than just providing syntax examples.
